# MurFlame Launcher

Современный лаунчер Minecraft для Windows с поддержкой **лицензии (Microsoft)** и **офлайн-режима** (пиратка).

## Возможности

- **Аккаунты**: вход через Microsoft (лицензия) или офлайн-ник
- **Версии**: загрузка релизов с официального манифеста Mojang
- **Запуск игры**: автоматическая установка клиента, библиотек и ресурсов
- **Настройки RAM**: слайдеры Xms / Xmx, дополнительные JVM-аргументы
- **Интерфейс**: темы MurFlame / тёмная / светлая, акцентный цвет, прозрачность
- **Java**: автоопределение установленных JDK

## Требования

- Windows 10/11
- Node.js 18+
- Java 17+ (для Minecraft 1.17+)

## Установка и запуск

```bash
cd MurFlameLauncher
npm install
npm run dev
```

Команда `npm run dev` запускает Vite и автоматически открывает окно Electron с подключённым API.

### Сборка .exe (обычное приложение для Windows)

1. Установите зависимости (один раз):

```bash
npm install
```

2. Соберите установщик:

```bash
npm run build
```

3. Готовый файл появится в папке:

`release/MurFlame Launcher Setup 1.0.0.exe`

Запустите этот `.exe`, установите лаунчер — дальше можно пользоваться как обычной программой (ярлык на рабочем столе).

Для разработки без сборки:

```bash
npm run dev
```

## Использование

1. **Аккаунты** → добавьте офлайн-ник или войдите через Microsoft
2. **Версии** → выберите и при необходимости установите версию
3. **Настройки** → выделите RAM, выберите Java и тему
4. **Главная** → нажмите **ИГРАТЬ**

### Microsoft-авторизация

1. Нажмите «Microsoft (лицензия)» — откроется браузер
2. Войдите в аккаунт Microsoft / Xbox
3. Скопируйте код из URL (`code=...`) и вставьте в окно лаунчера

## Структура папки игры

По умолчанию: `%APPDATA%\.murflame`

Можно изменить в настройках.
